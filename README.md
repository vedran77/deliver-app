# Getting Started
### Prerequisites

  Expo-CLI
* npm
  ```sh
  npm i -g expo-cli
  ```

### Installation


1. Clone the repo
   ```sh
   git clone git@github.com:vedran77/deliver-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```js
   expo start
   ```
